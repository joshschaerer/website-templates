Accordion-Templates

An accordion component displays collapsible display of content within a limited space. By clicking on a header, the associated content expands and collapses, allowing for efficient use of space and improved user experience. This collection offers a range of accordion components, designed to meet the needs of various web design projects. Browse through the options and find the perfect fit for your website.

Basic accordion

A template for a very simple accordion.

Code

                        
                            <!--=============== BASIC ACCORDION ===============-->
                            <div class="accordion accordion--basic" id="accordion-basic">
                                <div class="accordion__item">
                                    <h3 class="accordion__header">
                                        <button
                                        class="accordion__title"
                                        id="accordion-basic-header-1"
                                        aria-controls="accordion-basic-panel-1"
                                        aria-expanded="false"
                                        data-accordion-header>
                                        <!-- TITLE -->
                                        </button>
                                    </h3>

                                    <section
                                        class="accordion__panel"
                                        id="accordion-basic-panel-1"
                                        aria-labelledby="accordion-basic-header-1"
                                        hidden>
                                        <!-- CONTENT -->
                                    </section>
                                </div>
                            </div>
                        
                    
                        
                            /*=============== VARIABLES CSS ===============*/
                            :root {
                              --hover-overlay-color: hsl(0, 0%, 2%, 0.1);
                            }
                            
                            /*=============== ACCORDION ===============*/
                            [data-accordion-header] {
                              display: flex;
                              justify-content: space-between;
                              padding: 0.5rem;
                              width: 100%;
                            }
                            
                            [data-accordion-header]:is(:hover, :focus),
                            [data-accordion-header][aria-expanded="true"] {
                              background-color: var(--hover-overlay-color);
                            }
                            
                            [data-accordion-header][aria-expanded="false"]::after {
                              content: "+";
                            }
                            
                            [data-accordion-header][aria-expanded="true"]::after {
                              content: "-";
                            }                            
                        
                    
                        
                            /*=============== ACCORDION ===============*/
                            const accordions = document.querySelectorAll("[data-accordion-header]");

                            // Add a click event handler to each accordion
                            accordions.forEach((accordion) => {
                            accordion.addEventListener("click", changeAccordions);
                            });

                            function changeAccordions(e) {
                            const target = e.target;
                            const parent = target.parentNode;

                            // Toggle the expanded state of the accordion
                            const expanded = target.getAttribute("aria-expanded") === "true" || false;
                            target.setAttribute("aria-expanded", !expanded);

                            // Toggle the hidden state of the accordion panel
                            parent.nextElementSibling.toggleAttribute("hidden");
                            }
